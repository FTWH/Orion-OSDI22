This folder contains the benchmarking scripts to compare prewarming to No prewarming

## Usage:

```
For prewarming:
./E2E_collect_with_prewarming.sh <Number_of_runs>

For no prewarming:
./E2E_collect_No_prewarming.sh <Number_of_runs>

recommended:
Number_of_runs=10
```
